# ZenOS

## The art of effortless computing

ZenOS is an operating system built on **Alpine Linux**.  
It’s designed to be **lightweight, minimal, and reliable** — an OS that *just works*.

### Features
- Based on Alpine Linux for simplicity and security
- Small footprint, optimized for performance
- Easy to use and quick to set up
- Focused on stability and practicality
